# 💼 Mi Portafolio

Portafolio web desarrollado con Next.js, TypeScript y Tailwind CSS.

## 🚀 Tecnologías

- **Framework:** Next.js
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Gráficos:** Recharts
- **Deployment:** Vercel

## ✨ Características

- 🎨 Diseño responsivo
- 📊 Visualización de datos interactivos
- ♿ Accesibilidad (WCAG 2.1)
- 🔍 SEO optimizado
